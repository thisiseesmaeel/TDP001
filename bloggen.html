Uppgift1 : Vecka 35

För att komma igång med Gitlab så fick jag skapa ssh-nyckeln vilket jag gjorde genom att köra "ssh-keygen -t rsa -C "GitLab" -b 4096" programmet på terminalen på min dator. Nästa steg var att kopiera den publika nyckeln från .ssh mappen där nyckeln var sparades.

Jag öppnade Gitlab och hittade Nytt projekt knappen på Gitlab dashboard för att skapa ett nytt projekt. Sedan använde jag "git init" kommandot på den katalagen jag har sparat min blogg i efter att jag gick in på servern. Sedan på samma katalag laddade jag ner mitt projekt från gitlab till den katalogen genom att köra:

git clone git@gitlab.liu.e:saysa289/uppgift1.git

Efter den la jag till en fil med "README.md" namnet via "git add README.md", sedan skrev jag ett kommentar via "git commit -am "add README". Till slut pushade jag repositoryn tillbaka till gitlab genom att köra ,"git push -u origin master" så hela innehållet syns på gitlab med. Nu för att mina andra filer ska synas på Gitlab, öppnade jag TDP001 katalogen, sedan: "git remote add origin git@gitlab.liu.se:saysa289/uppgift1.git" för att få tillgång till Gitlab och sedan "git add ." för att lägga till alla filerna i katalogen. I slutet så committade jag med git commit -m "first commit" och pushade med git push -u origin master

    [a-z]{2,5}\d{2,3}
